## Install
1. Clone this repo
2. npm install
3. Create a .env file to input your MySQL credentials 
4. npm run seed
5. npm start

## Technologies
* MySQL2
* Sequelize
* dotenv
* Express
* Javascript
* Node JS
* NPM

## Usage
Invoke from the command line with
> npm start
